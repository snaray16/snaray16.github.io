Industry Experience

Greeneridge Sciences Inc., Acoustician

  1. Developed an end-to-end machine learning pipeline for automatic detection, classification, and localization of bowhead whale calls, enabling analysis of 19 continuously deployed sensors over a two-month Arctic deployment. The pipeline involves three stages:

    Stage 1: A classifier that extracts whale calls with > 97% accuracy.

    Stage 2: An object detection algorithm with FasterRCNN architecture that boxes whale calls and classifies their type with 95% precision and 84% recall (Figure 1).

    Stage 3: A matching algorithm that matches calls across sensors and triangulates the whale call's source to, on average, within 200 meters of the original estimate obtained from the manual analyst (over a detection range of > 200 square kilometers).

  2. Developed a GUI for computation and visualization of underwater sound source bearings, enabling efficient contextualization of acoustic sources during data analysis (Figure 2).
  3. Conducted field work for sensor calibration, testing, and deployment/retrievals in the Arctic Ocean.
  4. Analyzed acoustic and environmental data for end-of-study report submitted to client.

Currently developing supervised and unsupervised machine learning methods for acoustic feature extraction of odontocete vocalizations to demonstrate species-level differences and vocal production learning in beluga whales using data-science and ML visualization techniques (t-SNE, UMAP, Topological Data Analysis, Class Activation Maps).
